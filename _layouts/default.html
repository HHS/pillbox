Opening the code of the Pillbox dataset

The Pillbox dataset is one of the largest databases of prescription drug information made available by the FDA and NLM. Pillbox for Developers is a resource for getting open access to the Pillbox data processing code, understanding the methodology, and participating in the development of the project.
